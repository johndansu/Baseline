FAQ

Common questions from engineering and security leaders.

How is this different from linting or static checks?

Baseline evaluates production policy as release gates with enforcement, remediation, approvals, and auditability in one system.

Can Baseline block deployments automatically?

Yes. You can configure deterministic gates in CI/CD so violations pause release promotion until required controls are met.

Does AI make autonomous changes?

No. AI assists with patch generation, but humans review and approve all policy-impacting changes before merge.

What do auditors get from Baseline?

Policy execution history, approval logs, exception records, and linked release artifacts that simplify audit preparation.
